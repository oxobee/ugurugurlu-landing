# ugurugurlu-landing

**Uğur UĞURLU – Yapay Zeka ile Görsel Üretim**

Professional AI-powered product photography and advertising video landing page for e-commerce sellers on Trendyol, Hepsiburada, Amazon, Etsy, ÇiçekSepeti, and Instagram.

🌐 **Live Site:** [ugurugurlu.net](https://ugurugurlu.net)

---

## Features

- ✅ **3 Languages:** Turkish / English / Arabic (with RTL support)
- ✅ **100% SEO Optimized:** hreflang, JSON-LD schema, sitemap.xml, robots.txt
- ✅ **GEO Targeted:** TR, DE, ES, US, NL, SA, AE
- ✅ **Interactive Chart:** Chart.js conversion rate visualization
- ✅ **Before/After Slider:** Custom drag-to-compare component
- ✅ **Video Section:** 5 vertical (9:16) + 1 horizontal (16:9) placeholders
- ✅ **Mobile First:** Fully responsive design
- ✅ **Floating WhatsApp Button**
- ✅ **Glassmorphism & Glow Effects**

## Tech Stack

- React 18 + Vite
- Tailwind CSS v4
- react-i18next (TR/EN/AR)
- react-helmet-async (SEO)
- Chart.js + react-chartjs-2
- Framer Motion
- lucide-react

## Quick Start

```bash
npm install
npm run dev
```
